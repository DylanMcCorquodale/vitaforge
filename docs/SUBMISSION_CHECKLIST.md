## Framework and data-store note

The instructions use Next.js and MongoDB as implementation guidance. VitaForge uses React with Vite, a Node REST API, SQLite locally, and Netlify Blobs when deployed. These equivalent choices cover the assessed full-stack concerns: React components, state management, authenticated APIs, persistent CRUD, automated tests, deployment, and documentation.
